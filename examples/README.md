# Examples

This directory contains usage examples for `@ng-states/core`.

## Available Examples

### [basic-usage.ts](basic-usage.ts)

Basic TypeScript usage demonstrating all core functions with type safety.

### [commonjs-usage.js](commonjs-usage.js)

CommonJS/Node.js usage example for JavaScript projects.

### [react-component.tsx](react-component.tsx)

React component example showing how to build a state and LGA selector with hooks.

### [vue-component.vue](vue-component.vue)

Vue 3 component example using Composition API for state and LGA selection.

## Running Examples

### TypeScript Examples

```bash
# Install ts-node if you haven't already
npm install -g ts-node

# Run TypeScript examples
ts-node examples/basic-usage.ts
```

### JavaScript Examples

```bash
# Run JavaScript examples
node examples/commonjs-usage.js
```

### Framework Examples

The React and Vue examples are meant to be copied into your respective projects. They require the framework dependencies to be installed in your project.

## Contributing Examples

Have a useful example? Feel free to contribute! Examples should:
